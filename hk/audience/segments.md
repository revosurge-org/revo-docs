# 受眾細分

**對象：** UA 經理、增長團隊、廣告系列營運

受眾細分允許您根據行為訊號定義用戶群體，然後使用這些細分進行廣告系列定向。

## 本文內容

- 細分詳情與欄位
- 注意事項與限制(目前版本)

## 甚麼是受眾細分？
**受眾細分**是儲存的定義，將符合特定條件的用戶分組(如點擊廣告、訪問網站的用戶等)。

細分通常用於：
- 再定向已互動用戶
- 排除低質素用戶
- 建立基於興趣或行為的受眾

## 細分的用途
細分可用於廣告系列定向([**AdWave → 引導式廣告系列設定**](/hk/adwave/guided-campaign-setup) 的步驟 3)，當 UI 允許選擇受眾標籤/細分時。
