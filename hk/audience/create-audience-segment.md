# 建立受眾細分(目前 UI)

**對象：** 廣告系列營運、增長團隊

本節描述目前可用的細分建立 UI。

## 本文內容
- 輸入細分元數據
- 選擇匹配類型
- 新增規則(行為條件)
- 執行與驗證
- 儲存並使用

## 1) 輸入細分元數據
在 **AdWave → 受眾** 中，點擊 **建立受眾** 並填寫：
- **名稱**：列表中顯示的細分名稱
- **描述**(可選)：細分的用途

## 2) 選擇匹配類型(AND / OR)
定義規則時，選擇多個條件的組合方式：
- **滿足全部條件 (AND)**：用戶必須滿足每條所選規則  
- **滿足任一條件 (OR)**：用戶滿足任一規則即可  

## 3) 新增行為規則(條件)
在 **Behavioural** 下，選擇一個或多個條件(根據目前 UI 可用性)，例如：
- **廣告曝光** — 看過您廣告的用戶  
- **廣告點擊** — 點擊過您廣告的用戶  
- **網站訪問** — 訪問過您網站的用戶  

> 如 UI 支援展開每條規則(下拉箭頭)，可用其選擇時間視窗或額外參數。

## 4) 檢視摘要與預估規模
右側 **摘要** 面板用於驗證：
- 組合邏輯(AND/OR)
- 預估受眾規模(在系統可計算時)

點擊 **執行**(如有)以產生預估並驗證配置。

## 5) 儲存細分
儲存後，您的細分將出現在 **受眾細分** 列表中，包含：
- **類型**(如 Behavioural)
- **來源**(如 User Defined)
- **規模**
- **使用中**(目前引用該細分的廣告系列數量)
- **狀態**(Active / Inactive)
