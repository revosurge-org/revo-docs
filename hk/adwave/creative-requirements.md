# 素材要求與範例

**對象：** 為 AdWave 準備素材的 UA 經理、廣告系列營運與增長團隊。

> **提示：** 建議在投放前先準備至少 **8–10 組素材（creative sets）**。AdWave 會用算法自動測試不同素材、把預算集中到表現最好的幾組、淘汰開始疲勞的素材。素材越多樣，算法優化空間越大，獲客成本（CPA）越低。

## 為什麼建議 8–10 組 {#why-8-10-sets}

- **算法靠多樣素材學習。** 素材越多樣、差異越明顯，算法學得越快、越準。
- **更低的獲客成本（CPA）。** 可優選的空間越大，同樣預算買到的轉化越多。
- **抵抗素材疲勞。** 多組素材可輪替替換，維持長期投放成效。
- **更快鎖定方向。** 一次測多組，更快找出最有效的組合。

**AdWave 怎麼測試：**

- **一次只變一個變量**，這樣才讀得出是什麼在起作用。
- **逐層優化。** 跑出勝出促銷後，依「促銷 → 角度 → 視覺」往下測。
- **持續換新。** 素材疲乏（點擊率／轉化下滑）時，每 2 週補 2–3 組替換。

## 素材組的定義 {#what-is-a-creative-set}

「多樣素材組」是指在以下幾個關鍵維度上刻意做出差異，讓算法有東西可以比較。建議先挑**貴站已驗證、轉化最好的幾個變體**來準備：

1. **促銷（Offer）** — iGaming 轉化的核心變數：如 100% 首存紅利、50 次免費旋轉、小額無流水要求體驗金、2% 輸返、VIP 方案、限時活動等。
2. **主題** — 不同內容切角：老虎機、Crash、真人娛樂場、體育投注、撲克等。
3. **賣點／訊息** — 不同訴求角度：大獎金額、提款快、安全合規、新用戶專屬、社群／真實感、緊迫感（倒數／限量）。
4. **設計／視覺風格** — 如真人荷官／美女照片、遊戲動畫、中獎畫面、體育明星等。
5. **格式** — 目前支援四種版位：**Pop、Push、Display、Native**——各有不同呈現與表現（[規格見下文](#supported-placements-specs)）。
6. **受眾／本地化** — 語言、當地熱門玩法、當地支付方式、節慶（農曆新年、世界盃）、新客 vs 回流客。

## 第一輪需要準備的素材 {#what-to-prepare-for-round-1}

**以馬來西亞（MY）市場為例：**

| 維度 | 示例 |
| --- | --- |
| **促銷** | 首存 100% 紅利 · 獨家老虎機高達 2% 返水 |
| **主視覺** | 真人荷官／美女 · 遊戲畫面 |
| **訊息** | 緊迫感（倒計時） · 中獎爆獎 |
| **語言** | EN / ZH / MY（三語） |

**測試方案：**

| 版位 | 測試維度 | 促銷 A · 首存紅利 | 促銷 B · 獨家老虎機 | 組數 |
| --- | --- | --- | --- | --- |
| **Pop** | 主視覺 | 真人荷官／美女 · 遊戲畫面 | 真人荷官／美女 · 遊戲畫面 | 4 |
| **Display** | 尺寸覆蓋 | 300×250 · 300×100 · 320×50 | 300×250 · 300×100 · 320×50 | 2 |
| **Push** | 文案角度 | 緊迫感·倒計時 · 中獎爆獎·社群證明（任一促銷通用） | | 2 |

**8 組素材，共需產出約 28 項素材：**

| 版位 | 概念數 | 本地化 | 產出 |
| --- | --- | --- | --- |
| Pop | 2 促銷 × 2 主視覺 | 多語自適應 | 4 個 URL |
| Push | 2 個角度 | × 3 語言 | 6 組 |
| Display / Native | 2 促銷 × 3 尺寸 | × 3 語言 | 18 個 |
| **合計** | **8 個素材概念** | **3 語言（EN / ZH / MY）** | **約 28 項素材** |

## 支援的版位與規格 {#supported-placements-specs}

我們支援以下四種版位，請按規格提供素材。

> **提示：** Pop 與 Display 優先做動態版本。動態素材（Display 提供 GIF）通常更吸睛，也更能帶動點擊與轉化。

| 版位 | 格式／內容 | 規格 |
| --- | --- | --- |
| **Pop** | 落地頁網址（URL） | 每個促銷提供一個支援多語自適應的落地頁網址 |
| **Push** | 品牌 Logo + 圖片（desktop 廣告）+ 標題 + 描述 | 品牌 Logo：JPG/PNG，192×192，≤ 200 KB · 圖片：JPG/PNG，360×240，≤ 720 KB · 標題 ≤ 30 字 · 描述 ≤ 40 字 |
| **Display** | GIF / JPG / PNG，≤ 500 KB | 尺寸：300×250、300×100、320×50 |
| **Native** | JPG / PNG，≤ 500 KB（靜態） | 尺寸：300×250 · 融入內容（圖 + 標題 + 描述） |

### Pop

- **格式：** 落地頁網址（URL）——Pop 呈現為落地頁，只需提供 URL。
- **每個促銷：** 1 個 URL，多語自適應（EN / ZH / MY）。
- 每個促銷準備 **2 個主視覺**（如真人荷官／美女、遊戲畫面），分開測試。
- 建議使用**動態**視覺。

### Push

- **品牌 Logo：** 192×192 · JPG/PNG · ≤ 200 KB
- **圖片：** 360×240 · JPG/PNG · ≤ 720 KB
- **標題** ≤ 30 字 · **描述** ≤ 40 字（表情符號不計入上限）。
- 常用角度：緊迫感／限時倒計時，以及中獎爆獎／社群證明。

### Display

- **格式：** GIF / JPG / PNG · ≤ 500 KB
- **尺寸：** 300×250、300×100、320×50——每個促銷出齊三個尺寸。
- 小尺寸重新排版，不要把大尺寸等比縮小。
- 建議使用**動態 GIF**。

### Native

- **格式：** JPG / PNG（靜態） · ≤ 500 KB
- **尺寸：** 300×250
- **風格：** 融入內容——圖 + 標題 + 描述。

## 合規 {#compliance}

所有素材必須含**責任博彩與年齡限制字樣**（18+／21+ 視地區而定），並附**條款（T&Cs）**連結。實際文案請以貴司核准版本為準。
