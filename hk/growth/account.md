# 建立 RevoSurge 帳戶

**對象：** 管理員 / 主帳戶用戶、帳戶擁有者、營運

## 甚麼是 RevoSurge 帳戶？
RevoSurge 帳戶代表您在平台上的業務，是以下內容的容器：
- 法定公司身份
- 計費與充值（AdWave 及 DataPulse 共享錢包）
- 用戶、角色及權限
- 您要推廣的產品（網站/應用）
- 廣告系列、分析及財務餘額

## 前提條件
準備：
- 法定公司名稱
- 註冊營業地址及國家
- 主要聯絡人姓名及電郵
- 可選計費聯絡人（建議）

## 建立或啟用帳戶
1. **註冊或接受邀請**
   - 使用 RevoSurge 代表提供的註冊連結，或
   - 如公司帳戶已存在，接受邀請電郵。

2. **填寫公司資料**
   - 法定公司名稱
   - 註冊國家
   - 營業地址
   - 主要聯絡人（姓名 + 電郵）

3. **接受在線協議**
   您的點擊接受即視為數碼簽署。
   接受後，如必填資料完整，帳戶狀態可變為**就緒**。

## 新增團隊成員及角色
內建角色：
- **管理員（主帳戶）**：完全存取（包括用戶/角色管理）
- **廣告主管**：管理產品、廣告系列、素材及報表（分配範圍）
- **數據分析**：管理數據資產並檢視廣告系列/報表（分配範圍）
- **財務**：檢視餘額、發票及財務報表

邀請方式：
1. 前往 **帳戶 → 成員**
2. 點擊 **邀請成員**
3. 輸入電郵 + 角色 + 可選留言
4. 點擊 **發送邀請**

注意：
- 您可稍後更改角色、停用用戶或重新發送邀請。
- 每個帳戶至少保留**一名管理員/主帳戶**（最後一名管理員不可移除）。
