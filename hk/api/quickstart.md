# API 快速入門

**對象：** 開發者、技術整合方、BI 工程師

RevoSurge API 支援您自動化工作流程，並將報表及事件整合到自有系統中。

## 本文內容
- API 能做甚麼
- 典型整合模式
- 開始前（認證）

## API 能做甚麼
常見用例包括：
- 發送**伺服器到伺服器 (S2S) 事件**
- 拉取報表匯總（曝光、點擊、支出、轉化）在可用時
- 管理金鑰及存取（視乎控制台能力）

## 典型整合模式
- 每日匯出到數據倉庫用於報表
- 充值/交易的伺服器事件
- 建立用戶細分（如 API 支援）

## 開始前
您需要：
- API 金鑰（參見 **API 金鑰**）
- 正確的 Base URL 及端點參考（參見 **伺服器事件 API**）
