# RevoSurge API Key Management

**Audience:** Engineers, technical integrators, Admin teams

## What you can do with the API Keys
API Keys from RevoSurge enable authentication for API requests.

### API Key Management
- Create API keys for each product
- Rotate keys regularly for security
- Revoke keys if compromised or no longer needed

### Create API Key
To create an API key:
1. Log in to your RevoSurge account.
2. Navigate to the API Keys section in the Admin panel.
3. Click "Create API Key" and follow the prompts to generate a new key.
4. You must assign the key to specific product, multiple keys can be assigned to the same product.
5. An alias name for the key (e.g., "My Integration Key") helps identify its purpose.

### Use API Key
Once created, the API key is displayed. Copy it and store it securely. Avoid exposing it in public repositories.

### Disable API Key
To disable an API key:
1. In the API Keys section, locate the key you want to disable.
2. Click the "Disable" button next to the key.
3. Confirm the action when prompted.

### Delete API Key
To delete an API key:
1. In the API Keys section, locate the key you want to delete.
2. Click the "Delete" button next to the key.
3. Confirm the action when prompted.
