# 受众细分

**面向：** UA 经理、增长团队、广告系列运营

受众细分允许您根据行为信号定义用户群体，然后使用这些细分进行广告系列定向。

## 什么是受众细分？
**受众细分**是保存的定义，将符合特定条件的用户分组（如点击广告、访问网站的用户等）。

细分通常用于：
- 再定向已互动用户
- 排除低质量用户
- 创建基于兴趣或行为的受众

## 细分的用途
细分可用于广告系列定向（[**AdWave → 引导式广告系列设置**](/cn/adwave/guided-campaign-setup) 的步骤 3），当 UI 允许选择受众标签/细分时。
