# 创建受众细分（当前 UI）

**面向：** 广告系列运营、增长团队

本节描述当前可用的细分创建 UI。

## 本文内容
- 输入细分元数据
- 选择匹配类型
- 添加规则（行为条件）
- 运行与验证
- 保存并使用

## 1) 输入细分元数据
在 **AdWave → 受众** 中，点击 **创建受众** 并填写：
- **名称**：列表中显示的细分名称
- **描述**（可选）：细分的用途

## 2) 选择匹配类型（AND / OR）
定义规则时，选择多个条件的组合方式：
- **满足全部条件 (AND)**：用户必须满足每条所选规则  
- **满足任一条件 (OR)**：用户满足任一规则即可  

## 3) 添加行为规则（条件）
在 **Behavioural** 下，选择一个或多个条件（根据当前 UI 可用性），例如：
- **广告曝光** — 看过您广告的用户  
- **广告点击** — 点击过您广告的用户  
- **网站访问** — 访问过您网站的用户  

> 如 UI 支持展开每条规则（下拉箭头），可用其选择时间窗口或额外参数。

## 4) 查看摘要与预估规模
右侧 **摘要** 面板用于验证：
- 组合逻辑（AND/OR）
- 预估受众规模（在系统可计算时）

点击 **运行**（如有）以生成预估并验证配置。

## 5) 保存细分
保存后，您的细分将出现在 **受众细分** 列表中，包含：
- **类型**（如 Behavioural）
- **来源**（如 User Defined）
- **规模**
- **使用中**（当前引用该细分的广告系列数量）
- **状态**（Active / Inactive）
