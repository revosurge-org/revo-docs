# 引导式广告系列设置

**面向：** 在 AdWave 投放广告的 UA 经理、广告系列运营与增长团队。

引导式广告系列设置是一个单页、逐步进行的流程，带你在同一处完成从创建产品到上线投放（含素材）的全过程。

## 前提条件 {#prerequisites}

广告系列创建**受数据配置限制**。在 AdWave 引导页（**接入你的数据**）上，*创建广告系列* 会一直锁定，直到你完成步骤 1–3。开始前请确保：

- **已创建产品。** 创建**产品**（你要推广的应用或网站），并填写其域名、追踪器 ID 与充值汇率。
- **Web 追踪器已激活。** 在 Web 追踪器收到首个事件之前，产品处于 **Inactive（未激活）** 状态；只有**已激活**的产品才能在广告系列流程中选择。参见 [安装 Web 追踪器](/cn/tracking/web-tracker/install)。
- **转化事件已追踪。** 你用于优化的目标事件（如 Register、Deposit）必须已通过追踪流入。参见 [追踪 → 概述](/cn/tracking/overview)。
- **钱包已充值。** 你的**钱包**余额充足。参见 [充值与钱包](/cn/growth/funding-wallet)。
- **素材已就绪。** 素材已按规格准备好。参见 [素材要求与范例](/cn/adwave/creative-requirements)。

> **提示：** 你可在**管理产品**中查看产品状态——**Tracker Status（追踪器状态）** 列显示 Active / Inactive，*Data Setup（数据配置）* 显示有多少事件处于激活状态。

## 1) 产品 {#_1-product}

选择你要推广的**产品**。每张卡片显示产品名称及其 URL。

> **注意：** “未激活的产品需先激活才能投入使用。” 若所需产品尚未激活，请先激活，再返回本流程。

## 2) 目标与广告格式 {#_2-objective-ad-format}

**广告系列名称**

输入用于报表的可读名称。AdWave 建议一套命名格式以保持广告系列一致：

> 建议格式：**地区 + 促销 + 广告格式 + 开始日期**——例如 `IN_Register_Display_20260126`。

**目标事件**

选择本广告系列要优化的转化事件。该事件用于衡量转化并计算广告系列的 **CPA**。

- **Register**
- **Deposit**

**广告格式**

选择一种格式。每种格式的素材要求不同（见 [步骤 6](#_6-creative)），建议出价也不同：

- **Pop** — 以覆盖层或新窗口形式出现的广告。极为吸睛；适合促销与以转化为目标的广告系列。
- **Push** — 直接推送到用户设备的通知式广告。适合再互动与持续触达用户。
- **Display** — 放置在固定位置（如侧边栏或页首）的横幅式广告。适合强曝光与广泛的品牌认知。
- **Native** — 无缝融入内容信息流的广告。干扰更小、更具吸引力；适合追求更高点击率。

## 3) 目标地区 {#_3-target-geo}

搜索并选择要定向的国家或地区。每个广告系列最多可添加 **20 个**地区。

> **提示：** 保持地区与素材语言一致。若你定向多个需要不同语言的市场，请为每个市场准备对应的素材（或素材语言）。参见 [步骤 6：素材](#_6-creative)。

## 4) 出价与日预算 {#_4-bid-daily-budget}

**出价**

- **出价策略：** **CPM**（每千次展示成本）。
- **最高 CPM 出价：** 你愿意为每千次展示支付的最高价格。

AdWave 会显示随广告格式而变化的**建议最高出价**（例如撰写本文时 Display ≈ \$0.04、Native ≈ \$0.03、Push ≈ \$0.11、Pop ≈ \$0.11）。

> **注意：** “这是我们为本广告系列建议的最高出价。竞价中的实际出价可能低于此上限。”

**日预算**

以美元（USD）设置每日花费上限。

> **提示：** 将日预算最低设为**目标出价的 20 倍**，以确保投放稳定、结果有意义。这样算法每天才有足够数据来学习并有效优化效果。

## 5) 排期 {#_5-schedule}

- **开始日期：** 默认为现在；也可安排未来开始。
- **结束日期：** 选择一个结束日期，或使用**快速选择周期**从开始日期起设置 **7d / 14d / 30d / 60d / 90d** 的时长。

## 6) 素材 {#_6-creative}

> **注意：** 请先在 [步骤 2](#_2-objective-ad-format) 选择广告格式。素材字段会随格式而变化。

点击 **添加素材（[格式]）** 以新增一个素材块。你可以**一次上传多张图片——最多 20 张**。

对每个素材：

- **素材语言：** 设置语言（如 EN）。勾选 **“本广告系列的所有素材都使用此语言”** 可应用于每个素材。
- **目标 URL：** 选择落地页。勾选 **“本广告系列所有素材共用一个 URL”** 可复用单个 URL，或使用 **添加落地页** 注册新的 URL。

各格式专属字段：

| 格式 | 素材 | 规格 |
| --- | --- | --- |
| **Pop** | 仅目标 URL | 无需图片——Pop 直接呈现你的落地页 |
| **Push** | 图标 + 主图 + 标题 + 描述 | 图标：JPEG/PNG/JPG，192×192，≤ 200 KB。主图：JPEG/PNG/JPG，≤ 720 KB，尺寸 492×328 / 360×240 / 720×480 |
| **Display** | 主图 + 目标 URL | JPEG/PNG/JPG/GIF，≤ 500 KB，尺寸 300×250 / 320×50 / 728×90 / 468×60 / 160×600 / 300×100 |
| **Native** | 主图 + 标题 + 描述 + 目标 URL | JPEG/PNG/JPG，≤ 500 KB，尺寸 300×250 / 300×100 / 320×50 / 728×90 / 468×60 / 160×600 |

> **提示：** 关于完整的素材指引、促销／角度测试与范例，参见 [素材要求与范例](/cn/adwave/creative-requirements)。

## 发布 {#launch}

没有单独的审核步骤。当每个部分都完成后，点击（右上角的）**发布** 即可激活或排期广告系列。

**发布前检查清单：**

- ✅ 已选择产品（且已激活）
- ✅ 广告系列名称遵循建议格式
- ✅ 已设置目标事件（Register / Deposit）
- ✅ 已选择广告格式
- ✅ 至少添加一个目标地区（≤ 20）
- ✅ 已设置最高 CPM 出价与日预算（预算 ≈ 出价的 20 倍）
- ✅ 已排期开始（及结束/周期）
- ✅ 素材已按规格添加，并设置语言与目标 URL

## 上线后：需监控的内容 {#after-launch-what-to-monitor}

**投放**

- 展示、花费、CPM
- 胜率（如可用）
- 地区分布 vs 预期

**互动**

- 点击、CTR
- 极高展示 + 几乎零点击 → 排查库存质量

**成效**

- 转化（Register / Deposit）
- CPA / ROAS（如已连接）
