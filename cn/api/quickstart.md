# API 快速入门

**面向：** 开发者、技术集成方、BI 工程师

RevoSurge API 支持您自动化工作流，并将报表和事件集成到自有系统中。

## 本文内容
- API 能做什么
- 典型集成模式
- 开始前（认证）

## API 能做什么
常见用例包括：
- 发送**服务器到服务器 (S2S) 事件**
- 拉取报表汇总（曝光、点击、支出、转化）在可用时
- 管理密钥和访问（取决于控制台能力）

## 典型集成模式
- 每日导出到数据仓库用于报表
- 充值/交易的服务器事件
- 构建用户细分（如 API 支持）

## 开始前
您需要：
- API 密钥（参见 **API 密钥**）
- 正确的 Base URL 和端点参考（参见 **服务器事件 API**）
